Este proyecto es una calculadora básica hecha en Python. Permite realizar operaciones simples como suma, resta, multiplicación y división.

El programa está dividido en dos archivos:

Operaciones.py: contiene las funciones de cada operación.
main.py: se encarga de pedir los datos al usuario y mostrar el resultado.

La calculadora pide dos números y luego el usuario elige qué operación quiere hacer. También incluye una validación para evitar la división entre cero.
